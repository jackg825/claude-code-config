# Architecture Debate Panel

## Round 1 (parallel, 3 agents)
1. **Advocate** — defend the proposal: strengths, precedents, evidence
2. **Critic** — challenge the proposal: risks, failure modes, alternatives
3. **Pragmatist** — real-world concerns: timeline, team size, tech debt

## Round 2 (sequential, 1 agent)
4. **Synthesizer** — integrate all three perspectives:
   - Points of consensus (all agree)
   - Points of contention (trade-offs required)
   - Recommended approach + key risks + mitigation strategies

## Constraints per Agent
- Output limit: 150 lines
- Must include concrete evidence or examples; pure opinion not accepted
- Each argument must include a confidence level (1-10)

## Usage
Provide: architecture question, relevant constraints (scale, timeline, team)
